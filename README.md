📅 CampusSync - Zero Forget

「意志」に頼らず「仕組み」で勝つ、大学課題の自動同期システム。
AI開発環境を活用し、学生生活における最大の懸念事項である「課題の出し忘れ」を、個人の努力ではなくシステムの力で解決するために構築したプロトタイプである。

🛠 開発の動機 (Motivation)

現行の大学ポータルサイトを利用した課題管理には、以下の**「構造的な欠陥」**が存在する。

1. **確認コストの過多 (Overhead):** ポータルにログインし、複数の階層をクリックして締切を確認するという工程自体が、確認漏れを誘発する。
2. **転記時のヒューマンエラー (Manual Input Limit):** 手動でスケジュール帳やカレンダーアプリに書き写す際、日付の勘違いや入力忘れが避けられない。
3. **意志力への過度な依存:** 「後で確認しよう」という個人の意志に頼った管理体制では、多忙な時期に必ず破綻をきたす。

これまでの失敗経験から、**「自分の記憶力を信用しない」**ことを前提とし、**「情報の取得から登録までを全自動化する仕組み」**の実装に至った。

✨ 主な機能 (Features)

🔄 **1. ポータル自動巡回 (Automated Crawling)** [※開発検証中]
- **ロジック:** 大学のポータルサイトを定期的にバックグラウンドで巡回し、新着の課題やレポートの締切日を自動でスキャンする。
- **UX:** ユーザーは一度連携設定を行うだけで、自らサイトをチェックしに行く必要性がなくなる。

📅 **2. スケジュールアプリ同期 (Calendar Integration)**
- 抽出した締切情報を、普段利用しているカレンダーアプリ（Google Calendar等）へシームレスに登録する。
- 締切直前にはリマインド通知を飛ばし、物理的に「忘れる隙」を排除する。

🛠 **3. タスク優先度可視化 (Priority Visualization)**
- 締切までの残り時間を計算し、緊急度の高いものから順に整理して表示。やるべきことの優先順位を迷わせない。

💻 技術スタック (Tech Stack)

本プロジェクトは「Vibe Coding」の実践として、生成AIを用いたAI駆動開発 (AI-Driven Development) により、要件定義からプロトタイプ実装までのサイクルを高速化している。

- **Development Tool:** Bolt.new / StackBlitz (AI-Integrated IDE)
- **Frontend:** HTML5, Vanilla JavaScript, Tailwind CSS
- **Concept:** Mechanism-based Solution (仕組みによる解決)

🔮 Future Roadmap (今後の展望)

- **セキュアな自動化ロジックの確立:** セキュリティの壁を突破し、ユーザーの手を一切煩わせないバックグラウンド同期の実現。
- **チーム・フレンド共有機能:** 仲の良い友人と「課題リスト」を共有し、互いにリマインドを送り合えるソーシャル要素の追加。
- **LLMによる課題内容解析:** 課題の難易度や内容をAIが解析し、着手すべき推奨時間を提案するパーソナルアシスタント機能。
